# Moove API Contract (legacy)

Base URL: https://api.moove.fit

## Auth
- POST /users/login (request magic link / token email)
- GET /users/me (current user)
- POST /users/register
- POST /users/member/register

## Users
- GET /users/me
- GET /users/:id

## Content
- GET /videos
- GET /videos/:id
- GET /challenges
